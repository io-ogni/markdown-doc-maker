# Markdown Converter — Feature Backlog

> **Status: unverified idea list.** Generated by Lovable as suggestions, not yet
> vetted for feasibility, effort, or whether they fit the "free and safe" positioning.
> Everything here must stay **client-side only** (no uploads, no third-party calls
> with user content) to keep the privacy claim true. Some items below (e.g. "fetch
> images", "import from URL") would introduce network requests — flag those against
> the privacy promise before building.

## 🏆 The "wow" trio (highest-impact first)
1. **CodeMirror editor** — syntax highlighting + keyboard shortcuts + formatting toolbar
2. **Real image + hyperlink embedding in exports** (currently placeholders / stripped)
3. **Export themes** (Academic, Modern, Minimal, GitHub) with configurable page setup

---

## 1. Export fidelity
- Real image embedding in PDF/Word (currently just placeholders). Fetch images and embed as base64. ⚠️ *network request — check against privacy claim; consider local-file / data-URI images only*
- Clickable hyperlinks in exports (currently stripped to plain text)
- Syntax-highlighted code blocks with language detection (currently monospace-only)
- Task lists (`- [ ]` / `- [x]`) rendered as checkboxes

## 2. Markdown coverage
- Footnotes (`[^1]`)
- Definition lists
- Math / LaTeX (`$...$`, `$$...$$`) via KaTeX — high value for technical/academic users
- Mermaid diagrams rendered to images in exports
- Auto-linking of bare URLs
- Emoji shortcodes (`:smile:` → 😄)

## 3. Export options
- HTML export (standalone, styled)
- Plain text export
- Custom PDF page settings: size (A4/Letter), margins, orientation, font family/size
- Cover page / title / author metadata for Word & PDF
- Auto-generated table of contents from headings
- Page numbers & headers/footers in PDF

## 4. Editor experience
- Syntax highlighting in the textarea (CodeMirror or Monaco) — biggest UX leap
- Synchronized scroll between editor and preview
- Formatting toolbar (bold, italic, heading, link, list)
- Keyboard shortcuts (Cmd+B, Cmd+I, Cmd+K link, Cmd+S download)
- Auto-save to localStorage so refresh doesn't lose work
- Undo/redo history beyond browser native
- Find & replace

## 5. Multi-document / workflow
- Recent documents list (localStorage)
- Multiple export at once (PDF + Word + HTML, zipped)
- URL sharing — encode markdown into a shareable link
- Import from URL (fetch a raw `.md` from GitHub). ⚠️ *network request — check against privacy claim*
- Paste-from-Word / Google-Docs → convert HTML back to Markdown

## 💎 Delight
- Export themes (Academic, Modern, Minimal, GitHub) — pre-styled fonts/colors
- Word count, reading time, character count in the toolbar
- Print stylesheet for direct browser printing
- PWA / offline install
- i18n for non-English UI
